# pwa-weather
this is my first pwa

### Support Browser

####android

* chrome v59.0.3071.92
* QQ浏览器：支持缓存
* UC浏览器：支持缓存
* 魅族浏览器：支持缓存

除了chrome，其他都不支持添加到桌面图标

####ios 
* ios 10.2   safari 支持添加到桌面

* In order to test in your computer, you can install the extensions of web-server-for-chrome for chrome.
